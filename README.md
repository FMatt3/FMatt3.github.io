# FMatt3.github.io
Eine statische Webseite, die direkt über GitHub Pages gehostet wird. 

## Features
* **Responsive Design:** Optimiert für Smartphones, Tablets und Desktop-Bildschirme.
* **Modernes Styling:** Umgesetzt mit klarem HTML5 und CSS3 (optional: und JavaScript).
* **Blitzschnelle Ladezeiten:** Keine schwere Datenbank im Hintergrund, reines statisches Hosting.

---

## Technologien
Folgende Sprachen und Tools wurden für diese Webseite verwendet:
* **HTML5** – Für die Struktur der Seite.
* **CSS3** – Für das Design und Layout.
* **GitHub Pages** – Für das kostenlose und automatisierte Hosting.

---

## Projekt-Struktur
Hier ist eine kurze Übersicht, wie die Dateien in diesem Repository organisiert sind:

* `index.html` – Die Hauptseite (Startseite) der Website.
* `style.css` – Enthält alle Design- und Layout-Regeln.
* `assets/` – Ordner für Bilder, Icons oder Logos (falls vorhanden).
* `README.md` – Diese Dokumentation.
